This script allows to Create, Read, Update and Delete Projects.


1. Create

Using Create, you can add a Name, Description to the project. By default, the state of the project is set to "Open"


2. Read

Specify the project name and the script reads out the Name, Description, Owner, State and first level Children of the project

3. Update

Using this, Name, Description and State of the Project can be updated. Careful! On updating State to "Closed",
the project would no longer be visible under the project dropdown selection, it will be as good as deleted.

4. Delete 

The script sets the State of the specified project to "Closed", this project would no longer be visible under
Setup or under the Projects dropdown menu.
